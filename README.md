## كيفية الاستخدام

### 1. فتح الموقع
- افتح ملف `index.html` في أي متصفح

### 2. تخصيص المعلومات

#### تحديث معلومات التواصل:
افتح ملف `index.html` وقم بتعديل:

```html
<!-- رقم الهاتف -->
<p>+20 XXX XXX XXXX</p>

<!-- البريد الإلكتروني -->
<p>info@globalsecurity.com</p>

<!-- العنوان -->
<p>القاهرة، مصر</p>
```

#### تحديث روابط السوشيال ميديا:
```html
<!-- رابط الفيسبوك -->
<a href="https://www.facebook.com/YourPageName" target="_blank">

<!-- رقم الواتساب -->
<a href="https://wa.me/20XXXXXXXXXX" target="_blank">

<!-- رابط الإنستجرام -->
<a href="https://www.instagram.com/YourPageName" target="_blank">
```

#### تحديث رقم الواتساب في نموذج الاتصال:
افتح ملف `script.js` وابحث عن:
```javascript
window.open(`https://wa.me/20XXXXXXXXXX?text=${whatsappMessage}`, '_blank');
```
واستبدل `20XXXXXXXXXX` برقم واتساب شركتك (مع كود الدولة بدون +)

### 3. إضافة صور حقيقية للمنتجات (اختياري)
يمكنك استبدال الأيقونات بصور حقيقية:
1. أضف مجلد `images` في نفس مكان الملفات
2. ضع صور المنتجات في المجلد
3. استبدل `<i class="fas fa-video fa-5x"></i>` بـ `<img src="images/camera1.jpg" alt="كاميرا">`

## رفع الموقع على الإنترنت

### الطريقة 1: GitHub Pages (مجاني)
1. أنشئ حساب على GitHub
2. أنشئ repository جديد
3. ارفع الملفات
4. فعّل GitHub Pages من الإعدادات

### الطريقة 2: Netlify (مجاني)
1. سجل على Netlify.com
2. اسحب مجلد الموقع وأفلته على Netlify
3. سيتم رفع الموقع تلقائياً

### الطريقة 3: استضافة مدفوعة
- يمكنك رفع الملفات على أي استضافة تدعم HTML/CSS/JS

## ربط الموقع بصفحة الفيسبوك

بعد رفع الموقع:
1. انسخ رابط الموقع
2. اذهب إلى صفحة الفيسبوك الخاصة بك
3. اضغط على "تعديل معلومات الصفحة"
4. أضف رابط الموقع في خانة "الموقع الإلكتروني"
5. يمكنك أيضاً نشر رابط الموقع في منشور مثبت
